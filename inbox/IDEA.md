# IDEA.md — xx.idea.md 格式规范

## META
- 文档类型：报告 / 技术文档 / 其他
- 目标读者：（谁看，他们的背景是什么）
- 核心目的：读完之后读者应该知道什么 / 做什么
- 语气基调：正式 / 半正式 / 技术中性
- 输出格式：Word / Markdown / PPT大纲

## 规格约束
- 总字数：XXXX 字（±10%）
- 章节分配：如"引言10%，主体70%，结尾20%"
- 标题层级：最多 X 级（如 H1/H2/H3）
- 特殊排版要求：封面页 / 摘要框 / 编号列表等（无则填"无"）

- 【支撑】次要素材，用于佐证主干
- 【背景】可用可不用，按需取用
